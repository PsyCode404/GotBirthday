Welcome to the Game of Thrones themed Name Day celebration for Soumaya! This is a dark fantasy transformation of Birthday V4, featuring a medieval Westeros aesthetic with royal announcements, epic storytelling, and a journey through the Seven Kingdoms. A raven has arrived with a message...

## ⚔️ GoT Edition Features

* **Dark Fantasy Design:** A stunning dark medieval aesthetic featuring blacks, greys, royal gold, and deep red accents with glowing effects.
* **Atmospheric Background:** A dark shifting gradient with golden and crimson ember-like particles floating through the realm.
* **Five-Step Epic Journey:** A royal announcement from the Seven Kingdoms:
    1.  **The Raven's Arrival:** A mysterious gift sealed with the sigil of Westeros.
    2.  **The Golden Key:** An ancient key to unlock the secrets of the realm.
    3.  **The Chronicles of Westeros:** A leather-bound tome with parchment pages.
    4.  **Royal Proclamation:** Pages turn to reveal the Name Day celebration for Lady Soumaya.
    5.  **The Grand Finale:** An epic message honoring the birthday as a "Name Day" in true GoT fashion.
* **Advanced CSS & SVG Animations:**
    * **3D Page-Turning Effect:** Medieval book with realistic page flips.
    * **SVG Path Animation:** Golden decorative elements that draw themselves.
    * **Glow Effects:** Royal gold glows and shadows throughout.
* **Epic Music:** Designed for orchestral Game of Thrones style music (user must provide `got_theme.mp3`).
* **Medieval Typography:** Cinzel and Crimson Text fonts for authentic Westeros feel.
* **Fully Responsive:** Works perfectly on all devices while maintaining the dark fantasy atmosphere.

## 🚀 Getting Started

Bring the Seven Kingdoms to life with this setup.

### Prerequisites

* A modern web browser (Chrome, Firefox, Safari, Edge).
* An epic orchestral `.mp3` file for background music (Game of Thrones style recommended).
* (Optional) A dark castle or Iron Throne background image.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/Birthday-GoT.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd Birthday-GoT
    ```
3.  **Add your epic music:**
    Place your Game of Thrones style orchestral music in the folder and name it `got_theme.mp3`. The music should start automatically when the gift is clicked.
4.  **(Optional) Add background image:**
    For enhanced atmosphere, you can add a dark castle background image via CSS in `style.css`.
5.  **Open in Browser:**
    Simply open `index.html` in your browser to witness the Name Day celebration.

***

## 🎮 How to Customize

Make this Name Day celebration your own.

1.  **Recipient's Name:**
    * Open `index.html`.
    * Find the finale section (Step 5).
    * Change "Soumaya" to your recipient's name in the `<h2 class="finale-name">` tag.
        ```html
        <h2 class="finale-name">YourName!</h2>
        ```
2.  **Westeros Messages:**
    * In `index.html`, find the storybook pages (Step 3).
    * Edit the `.page-text` content to customize the royal proclamation.
    * Keep the medieval/GoT tone for authenticity.
3.  **Final Proclamation:**
    * Edit the `<p class="finale-wish">` text in Step 5.
    * Maintain the epic Westeros style (references to dragons, the North, the realm, etc.).
4.  **Color Scheme:**
    * Adjust CSS variables in `style.css` `:root` section.
    * Current palette: black, dark grey, royal gold (#d4af37), deep red (#8b0000).
